# MVP - Machine Learning & Analytics

Este projeto desenvolve um modelo de Machine Learning (ML) para prever a produtividade de soja e algodão no ano seguinte (Ex: dados de 2022 - previsão da produtividade de 2023), por Unidade da Federação (UF), utilizando dados públicos da PAM/IBGE. O estudo considera essas culturas no contexto da cadeia de produção do biodiesel.

O arquivo `dataset_ibge_soja_algodao.csv` contém uma cópia da base bruta obtida pela API pública do IBGE/PAM. Ele foi incluído para garantir a reprodutibilidade do notebook caso a API esteja temporariamente indisponível.
